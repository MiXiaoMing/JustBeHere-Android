# JustBeHere-Android 客户端+商户端

标语：一站式家庭服务

平台理念：非行业整合，只是线下服务线上化，达到便民，聚优，服务产品化，流程化等目的

平台优势：配套全 价格低  高品质服务


### 客户端主要分为服务、商城、订单、我的四个模块。

其中包含短信、统计、定位、推送、热修复等技术实现。
